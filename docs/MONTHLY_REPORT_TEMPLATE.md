# 月次レポート テンプレート

Looker Studio ダッシュボードと連動して、毎月運営会議に提出する数値レポートのフォーマット。

「目標 8,000 人来場」までの進捗を毎月可視化する。

---

## 使い方

1. 月初（毎月 1-5 日）に Looker Studio を開く
2. 期間を「先月 1 日〜末日」に設定
3. 以下のテンプレートに数値を埋める
4. 「考察」「次月アクション」を書き加える
5. 運営会議で共有

所要時間：**約 15 分**
